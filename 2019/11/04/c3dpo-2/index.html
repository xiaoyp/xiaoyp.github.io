Consistent factorization via canonicalization

对于NR-SFM来说，最大的挑战在于分解三维模型的视点的变化以及模型的变形带来的歧义。在这一部分中，作者提出了一种新的方法，直接去鼓励网络生成一致的模型。通常，令X0\mathcal{X}_0X0​作为通过网络获得的所有重建模型X(α;S)X(\alpha; S)X(α;S)的集合。如果网络能够将viewpoints与pose分解开，那么将不存在两种重建模型X,X′∈X0X, X' \in \mathcal{X}_0X,X′∈X0​满足X′=RXX' = RXX′=RX。也就是说，对于重建集合中的任意结构X，集合中除了其自身外不存在能够通过旋转操作得到X的结构。因此，论文中构建了标准化的函数Ψ:R3×K→R3×K\Psi :\mathbb{R}^{3 \times K} \to \mathbb{R}^{3 \times K}Ψ:R3×K→R3×K满足∀R∈SO(3),X∈X0,X=Ψ(RX)\forall R \in SO(3), X \in \mathcal{X}_0, X = \Psi(RX)∀R∈SO(3),X∈X0​,X=Ψ(RX)。

在C3DPO中，为保证能够将视点与物体的姿态有效地区分开，引入loss function为：

l2(X,R;Ψ)=1K∑k=1K∣∣X:,k−Ψ(RX):,k∣∣ϵ\mathcal{l}_2(X, R; \Psi) = \frac{1}{K}\sum_{k=1}^{K}||X_{:,k} - \Psi(RX)_{:,k}||_{\epsilon} l2​(X,R;Ψ)=K1​k=1∑K​∣∣X:,k​−Ψ(RX):,k​∣∣ϵ​

其中，R∈SO(3)R \in SO(3)R∈SO(3)是随机地一个旋转矩阵。

在网络训练过程中，输入为YnY_nYn​，首先通过网络Φ(Yn,v)\Phi(Y_n, v)Φ(Yn​,v)生成视点的参数θn\theta_nθn​以及姿态的参数αn\alpha_nαn​，在这一过程中，引入re-projection loss。除此之外，一个随机的旋转矩阵R^\hat{R}R^作用在生成的结构Xn=X(αn;S)X_n = X(\alpha_n; S)Xn​=X(αn​;S)上，之后，旋转后的结构R^Xn\hat{R}X_nR^Xn​被送入标准化神经网络Ψ\PsiΨ中，这一神经网络通过预测模型的系数α^n\hat{\alpha}_nα^n​达到抵消旋转矩阵R^\hat{R}R^的作用,生成预测的标准模型X^n=X(α^n;S)\hat{X}_n = X(\hat{\alpha}_n; S)X^n​=X(α^n​;S)，标准模型应该能够生成未经旋转的模型XnX_nXn​，在这一过程中，引入损失函数l2l_2l2​。这两个网络Φ\PhiΦ与Ψ\PsiΨ同时进行训练，最小化l1+l2l_1+l_2l1​+l2​。通过最小化上述损失函数，以达到有效分解视点与姿态的目的。

In-plane rotation invariance

根据旋转不变性，能够进一步为神经网络添加约束用于学习。令Y=ΠRXY = \Pi RXY=ΠRX为一个三维结构X的视图。将结构X按一个轴进行旋转，即将rz∈SO(2)r_z \in SO(2)rz​∈SO(2)作用在关键点上作为一个新的input，则这两个input输出的结果Φ(Y,v)=(α,θ)\Phi(Y, v) = (\alpha, \theta)Φ(Y,v)=(α,θ)以及Φ(rzY,v)=(α′,θ′)\Phi(r_zY, v) = (\alpha' , \theta')Φ(rz​Y,v)=(α′,θ′)，应该满足α=alpha′\alpha = alpha'α=alpha′。因此，修改reprojection loss function为：

l3(Y,v;Φ,S)=1K∑k=1Kvk∣∣rzYk−M(θ′)(α⊗I3)S:,k∣∣ϵl_3(Y, v; \Phi, S) = \frac{1}{K}\sum_{k=1}^{K}v_{k} ||r_zY_k - M(\theta')(\alpha \otimes I_3)S_{:,k}||_{\epsilon} l3​(Y,v;Φ,S)=K1​k=1∑K​vk​∣∣rz​Yk​−M(θ′)(α⊗I3​)S:,k​∣∣ϵ​

因此，最终的loss修改为l2+l3l_2+l_3l2​+l3​。
